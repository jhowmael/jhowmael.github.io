Estilo Web

O projeto Estilo Web foi desenvolvido como trabalho acadêmico para a disciplina de Engenharia de Software. Este sistema tem como finalidade auxiliar a administração de uma loja de roupas, controlando o registro de produtos, vendas, pagamentos e estoque. No sistema, o usuário pode cadastrar um produto, definindo todas as suas especificações, como cor, marca, tamanho, categoria e preço. Além disso, ele pode criar uma venda, vincular produtos e lançar os pagamentos. Para o desenvolvimento deste sistema, utilizei o CakePHP com o MySQL.
